## ANSI Color Chart Generator

Arquivo: `ansi-color-chart-generator/ansi-gen.py`

Não tem funções — é um script direto com três loops aninhados:

- Loop externo (`style`): alterna entre `0` (normal) e `1` (negrito/brilhante).
- Loop do meio (`fg`): percorre os códigos de cor de texto ANSI (30–37).
- Loop interno (`bg`): percorre os códigos de cor de fundo ANSI (40–47).

Para cada combinação, monta o código `style;fg;bg`, imprime o texto formatado com esse código (`\033[{code}m {code} \033[0m`) e reseta a formatação em seguida. Ao final de cada linha de `bg` pula uma linha, e ao final de cada bloco de `style` pula uma linha extra para separar a seção normal da em negrito.

## Number Guessing Game

Arquivo: `number-guessing-game/number_guessing_game.py`

| Função | O que faz |
|---|---|
| `ask_int(prompt, low, high)` | Lê um input do usuário e garante que é um inteiro dentro de `[low, high]`, repetindo a pergunta em caso de valor inválido ou fora do intervalo. |
| `choose_difficulty()` | Mostra as opções de dificuldade (Fácil/Médio/Difícil) e retorna a tupla `(nome, tentativas_maximas)` escolhida pelo jogador. |
| `play_round()` | Executa uma partida completa: sorteia o número secreto entre `LOW` e `HIGH`, faz o loop de tentativas usando `ask_int` para o palpite, dá feedback de "muito alto"/"muito baixo" e retorna `True`/`False` conforme o jogador acerta ou esgota as tentativas. |
| `main()` | Loop principal do jogo: chama `play_round()` repetidamente, mantém o placar de vitórias/derrotas e pergunta se o jogador quer jogar de novo até ele responder que não. |

Constantes de suporte: `RESET/GREEN/RED/YELLOW/CYAN/BOLD` (códigos ANSI para colorir o feedback), `LOW`/`HIGH` (intervalo do número secreto) e `DIFFICULTIES` (mapa de opção → nome e nº de tentativas).
